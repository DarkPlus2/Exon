Complete control over your music experience with an extensive command library

play Music

Plays a song from YouTube, Spotify, or other supported platforms.

/play <song name or url>

Skips the currently playing song and moves to the next in queue.

/skip
next s

Displays the current music queue with pagination controls.

/queue [page]
q list

/playlist create <name> [songs]
pl create

Loads a saved playlist into the current queue.

/playlist load <name>
pl load

Frequently Asked Questions

Find answers to common questions about EXON

How do I add EXON to my server?

You can add EXON to your server by clicking the "Invite" button at the top of this page. This will redirect you to Discord's authorization page where you can select which server to add the bot to and customize permissions. You need "Manage Server" permissions to add bots.

What permissions does EXON need?

EXON requires the following permissions to function properly: View Channels, Send Messages, Embed Links, Connect, Speak, Use Voice Activity, and Add Reactions. The invite link will automatically request these permissions. For advanced features, you may need to grant additional permissions through your server settings.

How do premium subscriptions work?

Premium subscriptions are billed monthly and provide access to enhanced features. You can purchase premium for yourself (applies to all servers where you have DJ permissions) or for an entire server. Subscriptions auto-renew but can be canceled anytime. Payment is handled securely through Stripe.

Can I use EXON for commercial purposes?

Yes, our Server plan is designed for commercial use including streaming, radio stations, and public venues. However, you must comply with Discord's Terms of Service and ensure you have proper licensing for any copyrighted content played through the bot in commercial settings.

How do I report bugs or suggest features?

You can report bugs or suggest new features by joining our support server (linked at the top of this page). We actively monitor feedback and implement the most requested features in regular updates. Premium users get priority consideration for their suggestions.

Is EXON available in other languages?

Currently EXON supports English, Spanish, French, German, and Japanese. More languages are being added based on user demand. You can change the language in the bot's settings using the /language command (server admins only).
